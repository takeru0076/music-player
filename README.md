# 🎵 Music Player

このプロジェクトは、React、TypeScript、Vite、Tailwind CSS を使用して構築されたシンプルな音楽プレイヤーアプリケーションです。

## 🚀 使用技術

- React
- TypeScript
- Vite
- Tailwind CSS

## 📦 セットアップ方法

1. リポジトリをクローンします：

   ```bash
   git clone https://github.com/takeru0076/music-player.git
   cd music-player
   ```

2. 依存関係をインストールします：

   ``` bash
    npm install
    ```
3. 開発サーバーを起動します：

    ``` bash
    npm run dev
    ブラウザで http://localhost:5173 にアクセスして、アプリケーションを確認できます。
    ```

## 📁 ディレクトリ構成

music-player/
